<Start: FTS-162818-ACT-50, Requirement>
If the software activation succeeds, HeNB shall 
•	swap the running and not running attribute values
•	set the runningSwLifeCycleState to “activated”
•	set the notRunningSwLifeCycleState to “delivered”.
Note:  SCMS can read the running and not running attributes to determine the success or failure of the software activation.
