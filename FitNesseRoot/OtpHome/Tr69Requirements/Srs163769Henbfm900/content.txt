<Start: SRS-163769-HeNBFM-900, Requirement>
HeNB shall support sending attribute value change notifications (AVCN), Bulk change notification (BCN), Object Creation Notification and Object Delete Notification based on the eNB::FMProfile::enableRealTimeAVCN  settings configurable by the operator. (115149 LA5.0)
If enableRealTimeAVCN is true:
•	Real Time: All notifications are sent in real time.
•	Logged: Notifications are stored in the HENB and pulled only on request.
FRS Requirement: FRS-163769-R3          
	Affected DPM Area: HeNB-OAM, SCMS, Test          
	Release: SCLR1.1          
	Network Element: HeNB-OAM, SCMS, Test
