<Start: FTS-162818-ACT-60, Requirement>
If the software activation fails, HeNB shall 
•	set the notRunningSwDescrFileName to empty string
•	set the notRunningSwLifeCycleState to “unknown”
•	fall back to the previous software version.
Note:  SCMS can read the running and not running attributes to determine the success or failure of the software activation.
