<Start: SRS-162298-HeNBStrtUp-760, Requirement>
When the HeNB is required to register with the assigned HDM it shall first resolve the IPaddress of the assigned HDM by triggering a DNS Query to the internal DNS server using the attribute ManagementServer::URL.
The DNS Query shall be sent using untagged Ethernet frames.
